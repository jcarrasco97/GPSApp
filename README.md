## Descripción

Aplicación Android desarrollada en Java cuyo objetivo es obtener y mostrar la ubicación del dispositivo mediante GPS.

Este proyecto se centra principalmente en el uso de los servicios de localización de Android, la gestión de permisos en tiempo de ejecución y la correcta obtención de coordenadas de forma segura.

## Tecnologías utilizadas
- Java
- Android SDK
- Servicios de localización (GPS)

## Funcionalidades
- Solicitud y gestión de permisos de ubicación
- Obtención de la localización actual del dispositivo
- Visualización de las coordenadas obtenidas

## Aprendizajes
- Gestión de permisos dinámicos en Android
- Uso de los servicios de localización del sistema
- Buenas prácticas al trabajar con datos sensibles como la ubicación

## Description

Android application developed in Java that retrieves and displays the device's location using GPS.

The main goal of this project is to work with Android location services, handle runtime permissions, and safely obtain location data.

## Technologies
- Java
- Android SDK
- GPS / Location Services

## Features
- Request and manage location permissions
- Retrieve the device's current location
- Display GPS coordinates

## What I learned
- Handling runtime permissions in Android
- Working with location services
- Managing sensitive user data responsibly
